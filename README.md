# Razorpay Clone

A modern Razorpay-inspired landing page built with React, Vite, and Tailwind CSS. The project focuses on a clean fintech-style interface, responsive layout, and polished section-by-section presentation.

## Live Demo

View the deployed project here:

[https://razorpay-clone-amir-hamza-221386.vercel.app/](https://razorpay-clone-amir-hamza-221386.vercel.app/)

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Feather Icons

## Features

- Responsive Razorpay-style homepage
- Hero, product, CTA, testimonial, and footer sections
- Utility-first styling with Tailwind CSS
- Fast production builds with Vite
- Clean component structure for easy editing and expansion

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
